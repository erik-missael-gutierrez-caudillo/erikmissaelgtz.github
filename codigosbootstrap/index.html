Grupo de botones

< p> class="text-justify">Apliqué un div con clase .btn seguido de .btn-group para crear un grupo de botones.< / p>




<div class="btn btn-group">
<button class="btn btn-primary">Apple</button>
<button class="btn btn-primary">Samsung</button>
<button class="btn btn-primary">Sony</button>
</div>

Grupo vertical de botones

Apliqué a la class btn-group la clase vertical quedando de la siguiente manera class = "btn btn - group - vertical"

/* Código de los botones de arriba de mí */
<div class="btn-group btn-group-vertical">
<button class="btn btn-primary">Apple</button>
<button class="btn btn-primary">Samsung</button>
<button class="btn btn-primary">Sony</button>
</div>

Grupo de botones justificados

Para abarcar todo el ancho de la pantalla, use la .btn-group-justified clase:

<div class="btn-group">
<button class="btn btn-primary">Apple </button>
